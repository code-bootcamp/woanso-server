

# 📚 기획의도 
요즘 웹툰을 접할 수 있는 시스템은 많지만, 90년대생에게 하나의 소통창구였던 만화책 📚 서비스는 드물다.
친구들과 교실에서 만화를 공유하며 얘기했던 그 시절이 그리운 유저들에게 장소의 제약없이 직접 가지 않아도
”90년대 추억의 만화책”을 대여할 수 있는 서비스를 제공하고자 하였다.
관리자가 유저에게 만화책을 제공함으로써, 유저가 평소에 구하기 어려운 추억의 만화책을 대여할 수 있도록 한다. 
자유게시판을 통해 자유롭게 만화에 대한 얘기 등을 나누며, 유저들이 같은 추억을 공유할 수 있는 커뮤니티를 생성하였다.
